## Remaining Risks

- The knowledge-pack builder is a deterministic read-only contract over existing local fixtures; it does not add ingestion workers, persistence, provider calls, source facts, LLM calls, frontend UI, or generated output for non-cached assets.
- Eligible-not-cached assets return explicit non-generated `eligible_not_cached` pack states with no sources, citations, facts, chunks, recent developments, freshness hash, generated route, or reusable generated-output cache hit.
- Supported cached assets expose metadata and freshness hashes for existing `AAPL`, `VOO`, and `QQQ` fixture packs, but the response intentionally omits raw chunk text and full source document passages.
